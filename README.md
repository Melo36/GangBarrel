# GangBarrel

A puzzle game developed as part of the **Master Games** practical course. The topic of this project was "Chain Reaction".

We decided to use explosions as part of the chain reaction mechanics. You can blow up barrels and rocks which in turn can blow up other objects nearby. This is the main mechanic of the game.

There is also turn-based combat against enemy pirates.

The game also includes a level editor where players can create their own levels and browse levels created by other players.

---

## System Architecture & Tech Stack

* **Game Engine:** Unity 3D
* **Target Devices:** PC
* **Programming Language:** C#
